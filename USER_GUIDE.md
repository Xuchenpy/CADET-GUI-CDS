# CADET CDS - 用户指南

## 入门指南

### 先决条件
1. 已安装 Anaconda (例如，`D:\anaconda`)
2. 创建了 CADET-SHOWCASE conda 环境：
   ```
   conda create -n CADET-SHOWCASE python=3.10
   conda activate CADET-SHOWCASE
   pip install cadet-process numpy scipy matplotlib
   ```

### 启动应用程序
- **选项 A**：双击 `start.bat`
- **选项 B**：命令行：
  ```
  conda activate CADET-SHOWCASE
  python run_app.py
  ```

## 应用程序标签

### 1. 方法编辑器
创建和管理色谱方法参数：

- **柱标签**：设置柱尺寸（长度、直径、颗粒大小）
- **组分标签**：定义蛋白质组分和盐
- **梯度标签**：设计洗脱梯度步骤
- **SMA 标签**：配置 Steric Mass Action（空间质量作用）结合参数
- **扫描标签**：定义参数扫描范围
- **队列标签**：从多个方法创建批处理运行队列

**操作：**
- 保存：将方法存储到数据库
- 加载：选择并加载之前保存的方法
- 删除：从数据库中移除当前方法
- 新建：重置为默认参数

### 2. 方法运行
使用不同模式执行模拟：

- **从编辑器运行**：执行当前编辑器配置
- **运行保存的方法**：选择并运行存储的方法
- **扫描**：在定义的范围内运行参数扫描
- **队列**：按顺序批处理运行多个方法

结果完成后会自动保存到数据库。

### 3. 结果分析
浏览和分析模拟结果：

- **结果列表**：所有带时间戳的保存结果
- **色谱图**：带有缩放/平移的交互式图表
- **峰分析**：自动峰检测，包括高度、半峰宽（FWHM）、分辨率
- **叠加**：在一个图表上比较多个结果
  - 对齐模式：无、进样标记、主峰

**导出选项：**
- 导出 CSV：原始 UV/浓度数据
- 导出峰：峰分析报告
- 导出叠加 CSV：多结果比较数据

## 菜单栏

- **文件 > 退出**：关闭应用程序
- **工具 > 管理环境**：扫描 Python 环境
- **工具 > 检测包**：检查 CADET 包可用性
- **帮助 > 关于**：版本和功能信息

## 故障排除

| 问题 | 解决方案 |
|-------|----------|
| "CADET DLL not found" | 验证环境中是否安装了 CADET-Core |
| 模拟静默失败 | 检查 CADET-SHOWCASE 环境是否安装了 cadet-process |
| 无结果显示 | 点击结果分析中的"刷新"按钮 |
| 图表不更新 | 尝试调整窗口大小或切换标签 |

## 数据存储

所有数据都存储在应用程序目录中的本地 `cadet_cds.db`（SQLite）中。不需要网络连接或登录。
